== Description ==

Show the full or partial content from a page into your sidebar with a customizable widget.

== Installation ==

1. Upload `wp-page` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Widgets` area and drag&drop the `WP-Page` widget.
4. Configure the widget to do what you want.
